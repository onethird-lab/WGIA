WGIA v1.0 (CUDA)

Introduction

Welcome to WGIA V1.0, a powerful platform leveraging GPU technology for rapid and comprehensive omics data interaction analysis (including single omics and across multiple omics).

Explore the range of analyses available below:

1. Perform interaction analysis for SNP data.
2. Perform interaction analysis for CNV data.
3. Perform interaction analysis for DNAm data.
4. Perform interaction analysis for mRNA data.
5. Perform interaction analysis for miRNA data.
6. Perform interaction analysis for lncRNA data.
7. Perform interaction analysis for protein data.

Pre-installation

WGIA v1.0 (CUDA) is implemented using C language and CUDA. Before using it, please make sure the GPU is installed on the computer, and install CUDA programming environment first.

WGIA v1.0 manual

Method of use

When using local software WGIA v1.0 (CUDA) for analysis, the input data needs to be preprocessed. Please refer to the preprocessing R code download

WGIA v1.0 (CUDA, Command-line running):
Parameters:
-h Help information for this executable program.
-input The directory and name of the input file.
-input_group1 The directory and name of the input file of group1.
-input_group2 The directory and name of the input file of group2.
-input_cov The directory and name of the input file for covariates.
-output The directory and name of the output file.
-distribution The directory and name of the distribution file.
-output_distribution1 The directory and name of the distribution file of group1.
-output_distribution2 The directory and name of the distribution file of group2.
-model The model used for test.
-withCovariant Whether the model consider covariants.
-cutoff LD pairs with dprime greater than cutoff will be output.
-distribution The directory and name of LD frequency file.
-block The number of blocks for calculation which is an integer multiple of 1024 (default = 2048).
-thread The number of threads for calculation (default = 512).
Example:
 wgia.exe -analysis SNP_SNP_LD -model Dprime -input pre_matrix1.txt -output LD.txt -distribution LD_distribution.txt
