Quick start

This quick-start tutorial aims to assist users in navigating the WGIA platform, utilizing the provided example gene expression RNAseq FPKM-UQ dataset. This dataset comprises 96 breast cancer samples and 96 normal breast samples sourced from the TCGA, encompassing 8170 genes.

1. Download the example files including genome_data_matrix.zip and design_matrix.zip on the 'START_ANALYSIS' page.

2. Select the example zip files separately and click 'Submit' on the 'START_ANALYSIS' page.

3. After submitting the job, WGIA will direct you to the job status page. Each job is assigned a unique Job ID and a KEY, automatically generated. The Job ID allows for queue monitoring on the 'QUEUE' page, while the KEY permits result retrieval or sharing within a 30-day period post-processing.

4. The processing duration depends on the dataset's sample size and the number of markers. The status page automatically refreshes every 30 seconds, enabling users to track the progress of the interaction analysis through a process log while the data is being processed.

5. After the analysis completion, you can access the report page.

Input

Genome data matrix

Design matrix

The first column of the design matrix contains the sample names. The second and third columns denote labels for the case and control groups. For samples in the case group, the case column carries the label '1' and the control column, '0'. Conversely, for samples in the control group, the case column holds '0' while the control column holds '1'.

To expedite file uploads, WGIA exclusively accepts zip files containing the genome data matrix and design matrix, respectively.

Output

WGIA offers an example report page for users to review, providing access to all results, including visualizations, interaction tables, and download options.
